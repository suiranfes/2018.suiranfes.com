テーマ:
Super Crew
スパークル
Sparkle; 輝き、才気のきらめき

第66回翠巒祭のテーマ『Super Crew』は「スパークル(Sparkle)」と読むのですが、Sparkleには「輝き」という意味以外にも「才気のきらめき」という意味があります。

近年の翠巒祭では、新企画のSUIRAN STATION、校内展示への注力など、内面・外面共に磨きをかけて盛り上がることができました。そんな中、更なる盛り上がりのカギは、今まで十分に発揮されずにいた高高生の秘めたる才能にあると考えます。

今回の目標は、テーマの如く高高生一人一人が才能を十分に発揮し巧みに翠巒祭を作り上げる。そのために一人一人が活躍する場を増やすことです。

また、高高生一人一人が最高の(Super)乗務員(Crew)となり、全ての来場者が来て良かったと思えるような翠巒祭にする。そのために、前回同様、「おもてなし」の心を忘れないという意味も込められています。

翠巒祭とは

群馬県立高崎高校文化祭「翠巒祭」は、高高生が主体となって創り上げる県内最大級の文化祭です。

毎年開催されており、今年で66回目を迎えます。高高の一年間の行事の中でも最大の盛り上がりを見せ、ここ数年は1万人以上の方々がご来場くださっています。

翠巒祭の見どころは校門で来場したみなさんを迎えるアーチ、生徒全員の力を結集して作る巨大壁画、ミス高高コンテストや高高王などの楽しい企画が満載のイベントステージ、そして校舎内の一般展示。 高高生全員が作り上げた「翠巒祭」ぜひ心ゆくまでお楽しみください。

ご注意･ご案内

ご安全に配慮して

  • 会場内、およびシャトルバス内は大変混雑します。スリ・置き引き等のトラブルが発生する恐れがあります。十分ご注意ください。
  • 熱中症にご注意ください。水分補給をこまめにお願いします。
    模擬店にて飲み物をお買い求め頂けます。
  • 体調が悪くなった場合は、保健室(巨大壁画の裏 1F)にお越しください。 事態が急な場合は、周りの方のご協力をお願いいたします。
  • 公序良俗に反する行為を禁じます。発見した場合は実行委員及び教員から注意をさせていただきます。改善が見られない場合、ご退場いただく場合もございます。
  • 部室棟(理科棟)前にてベビーカーをお預かりします。校内展示を安全にご覧いただくためですので、ご理解とご協力をお願い申し上げます。

お問い合わせについて

  • 翠巒祭実行委員は、規定の黒いTシャツを着用しています。なにか不明なことがあればお気軽にお尋ねください。また、インフォメーションセンターが入口アーチ近くにあります。
  • 校内の地図はこちら、またはパンフレットの裏表紙にてご確認いただけます。

落とし物･お忘れ物に関して

  • 靴の落とし物･お忘れ物は、校舎内･1-9「本部」までお問い合わせください。
  • それ以外の落とし物･お忘れ物は、インフォメーションセンターまたは実行委員までお問い合わせください。

ごみ捨てに関して

  • 屋外のゴミステーションと、校舎内のごみ箱にてごみの回収を行っております。
  • ごみの分別回収を行っております。ごみの分別回収にご協力ください。
  • ゴミステーションの位置は地図をご確認ください。
  • ごみの分別に関しては、ゴミステーションにいる優しくてイケメンで笑顔の優しい総務班清掃課員がご案内します。

その他

  • 翠巒祭はすべて撮影可能です。公開する際は肖像権等にご配慮ください。
  • スリッパを用意しております。基本的に履物の心配はご不要です。お気軽にお越しください。
    ただし、SUIRAN STATIONでは用意されていますが、そのほかの体育館で行われるイベントではスリッパの用意がございません。必要な方はご持参ください。
  • 芝生を保護するため、指月庭の芝生への立ち入りはご遠慮ください。
